### // whoami

I've been shipping web and mobile products for the better part of a decade. I move comfortably across
the stack — from **Laravel** apps, APIs, and **MySQL/Postgres** schemas, to **React / Next.js**
front-ends, to **Flutter** and **React Native** mobile builds.

I love interfaces that feel alive — pixel-tight layouts, motion with intent, details you only notice on
the second look — and the complex systems humming underneath: state machines, data pipelines, auth
flows. Beauty on top, an engine below. That's the work.
